QUICK REFERENCE CARD

ESSENTIAL MOVEMENTS
───────────────────
h j k l       ← ↓ ↑ →
w b e         Word navigation
{ }           Paragraph navigation
gg G          Start/end of file
$ 0 ^         End/start/first char of line
f F t T ; ,   Find character on line


ESSENTIAL OPERATORS
───────────────────
d             Delete
y             Yank (copy)
c             Change
p P           Paste after/before
u Ctrl+r      Undo/redo
.             Repeat


ESSENTIAL MODES
───────────────
i a I A o O   Insert mode variants
v V Ctrl+v    Visual modes
Esc           Normal mode
:             Command mode


TEXT OBJECTS
────────────
iw aw         Inner/around word
i" a"         Inner/around quotes
i( a(         Inner/around ()
i{ a{         Inner/around {}
it at         Inner/around tags

TIPS & TRICKS

1. Combine operators with motions
   Example: d3w (delete 3 words), y$ (yank to end of line)

2. Use text objects
   Example: ci" (change inside quotes), dat (delete around tags)

3. Repeat with dot command
   Make a change, then use . to repeat it elsewhere

4. Visual block mode for columns
   Select columns with Ctrl+v, then I to insert on all lines

5. Macros for repetition
   Record with qa, stop with q, replay with @a

6. Marks for navigation
   ma to mark, `a to return

7. Relative numbering for easier counting
   Use :set relativenumber for precise navigation
